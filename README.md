## 如何使用这个仓库

1. 每个分类下按日期命名笔记文件，如 `2026-06-24-主题.md`
2. 笔记使用 Markdown 格式编写
3. 代码示例放在 `resources/code/` 目录
4. 图片等资源放在 `resources/images/` 目录
